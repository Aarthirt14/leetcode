## 🧠 Example Problem

### Valid Anagram

**Problem:**
Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, otherwise return `false`.

**Approach:**
Use a **hash map / frequency counter** to compare character counts.

**Time Complexity:** `O(n)`
**Space Complexity:** `O(n)`
